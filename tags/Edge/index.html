那些值得推荐的edge

Microsoft Edge

Microsoft Edge基于chromium开发，与Edge HTML相比，性能提升非常大，同时也带来了非常多chrome的特性，flags就是其中之一。下面是我认为有用的一些实验性功能（不保证在Win10上都能用，只在Win11测试过）

  • 1、开启平滑滚动：在新版edge中获得旧版edge平滑滚动，个人感觉改变不大

    ​ 直达➡ edge://flags/#smooth-scrolling

  • 2、开启多线程下载：这个绝对要开，下载速度提升很大 edge://flags/#enable-parallel-downloading

  • 3、开启Cleartype渲染：Windows在低分辨率下字体显示真的很糟糕，如果你是2K或1080p的屏幕建议试一试4K屏就没有必要了，原来的DC(directwrite)渲染足矣。 edge://flags/#edge-enhance-text-contrast

  • 4、关闭QUIC:实测可以提高谷歌系网站打开速度，当然国内一般没多少人用，开不开启无所谓（虽然我开了）

    edge://flags/#enable-quic –（默认好像已经禁用了）

  • 5、开启实验性JS：网页提速，似乎有点用 edge://flags/#enable-javascript-harmony

  • 6、开启自动HTTPS:安全第一，http自动转https，提高安全性 edge://flags/#edge-automatic-https

    P.S:这个网站更新速度会比较慢，但排版会优于学而思编程，所以请酌情选择，谢谢支持。
